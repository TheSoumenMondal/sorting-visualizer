# Sorting Visualizer

A lightweight sorting visualizer built with plain HTML, CSS, and JavaScript.

![Image](https://github.com/user-attachments/assets/4ba2950c-01fd-4f78-a021-adf8b4429bae)

## Features

- Visualizes Bubble, Selection, Insertion, Merge, and Quick Sort
- Real-time speed control while animation is running
- Pause and resume support
- Random array generation (12 to 20 bars)

## Currently Supported Algorithms

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

## Run

1. Open `index.html` in a browser.
2. Enter numbers (space-separated) or generate a random array.
3. Choose an algorithm and click Sort.

## Files

- `index.html` - UI structure
- `styles.css` - Styling and bar states
- `script.js` - Sorting logic and animation control
